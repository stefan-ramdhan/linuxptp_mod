# linuxptp_mod


## Description
This project is a fork of LinuxPTP (https://github.com/richardcochran/linuxptp). We are adding functionality for the calculation of Jitter as well as the implementation of the Egress Method defined in (https://www.ieee802.org/1/files/public/docs2016/as-aregev-measuring-recovered-clock-quality.pdf).

## Project Status
In Progress.
